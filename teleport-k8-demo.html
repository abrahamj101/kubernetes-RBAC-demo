The Problem We're Solving

Blast Radius: Every user has admin rights by default.

Security: One compromised account = full cluster takeover.

Audit: No containment or traceability.

Solution: Least-privilege access via Kubernetes RBAC

Identity via Certificates

  • 1 CSR with CN=deploy-user
  • 2 Signed by Cluster Root CA
  • 3 Embedded in kubeconfig
  • 4 K8s reads cert → Enforces RBAC

Certificate-Based Identity

No passwords. No tokens.
